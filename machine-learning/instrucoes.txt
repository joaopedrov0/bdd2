pra rodar em uma nova maquina voce tem que ter docker permitido pro usuario,
pra saber se está ok, rode "docker ps" deve rodar sem erros.
Caso haja erros, faça:
$ sudo groupadd docker
$ sudo usermod -aG docker $USER
$ newgrp docker
$ docker run hello-world
$ reboot
